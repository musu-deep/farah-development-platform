# البنية المعمارية

## الواجهة

React + Vite + TailwindCSS، مع Framer Motion للحركة وLenis للتمرير، وتقسيم الصفحات Lazy Loading.

## الخلفية

FastAPI مقسمة إلى Routers مستقلة:

- `auth`: التسجيل والدخول والخروج والجلسة.
- `consultations`: إنشاء الطلبات وعرضها وتحديثها.
- `admin`: مؤشرات التشغيل.
- `chat`: المساعد الذكي وسجل المحادثات.

## البيانات

MongoDB عبر Motor، مع فهارس للبريد والطلبات وسجل المحادثات. تستخدم الاختبارات قاعدة Mock غير متصلة بالشبكة.

## الأمان

JWT Access/Refresh داخل HttpOnly Cookies. جميع الأسرار متغيرات بيئة. لا يوجد اعتماد على بيانات الوصول الواردة في السجلات الأصلية.
